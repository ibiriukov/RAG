The OpenAI API allows developers to interact with models like GPT-4 or GPT-4o.
Common parameters:
- model: selects which model to use.
- temperature: controls randomness (0 = deterministic).
- max_tokens: limits output length.

Example (using Python):
from openai import OpenAI
client = OpenAI(api_key="your_api_key")
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello, how are you?"}]
)
print(response.choices[0].message.content)

Temperature in Open AI models controls randomness in generation.
0 = deterministic, higher values like 0.7 produce more diverse outputs.
